# Accessibility Helper Split Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Split KeyStats's accessibility-privileged event tap out of the main app into a long-lived helper (`KeyStatsHelper`) whose cdhash stays stable across main-app upgrades, so users authorize once and never revisit System Settings.

**Architecture:** Add a new Xcode target `KeyStatsHelper` that runs as a per-user LaunchAgent + MachServices. The helper bundle is prebuilt and committed as a zip payload inside the main app's `Resources/Helper/` directory along with a `HelperManifest.json`. On first launch the main app unpacks this payload to `~/Library/Application Support/KeyStats/Helper/KeyStatsHelper.app/`, registers a LaunchAgent, and talks to it over XPC (`com.keystats.app.helper`). Main app performs peer validation of XPC connections, holds one active session, decodes/translates events, and feeds `StatsManager`. Helper stays razor-thin: CGEventTap, button-role normalization, monotonic timestamps, and serialization only.

**Tech Stack:** Swift 5.9, AppKit, Foundation, NSXPCConnection / NSXPCListener, Security framework (SecCode*), launchd (launchctl bootstrap/bootout), CoreGraphics, Carbon/TIS, SPM (tests), Xcode 15+ (no `xcodebuild` for Swift builds from command line — use Xcode to compile, scripts only wrap archives).

**Target versions:** macOS 13.0 (`MACOSX_DEPLOYMENT_TARGET`), bundle id `com.keystats.app` (main), `com.keystats.app.helper` (helper), Mach service name and LaunchAgent label both `com.keystats.app.helper`.

## File Structure

### New files (main app)
- `KeyStats/InputEventDecoder.swift` — Keyboard decoding (`keyName(for:)`, `keyCodeMap`, `asciiKeyName`, layout cache) extracted from InputMonitor so both the legacy path and `RemoteEventProcessor` can use it. Owns TIS observer.
- `KeyStats/HelperSupervisor.swift` — Singleton: install / upgrade / uninstall of the helper bundle + LaunchAgent, canonical main-app path tracking, `isAuthorizedForAccessibility` accessor.
- `KeyStats/HelperXPCClient.swift` — NSXPCConnection wrapper: handshake, reconnect, invalidation, error plumbing.
- `KeyStats/RemoteEventProcessor.swift` — Implements `KeyStatsEventSinkProtocol`; takes helper payloads and feeds `StatsManager` (mirrors the consumer side of the old `InputMonitor`).

### New files (shared across both targets)
- `KeyStats/Shared/HelperProtocols.swift` — `KeyStatsHelperProtocol`, `KeyStatsEventSinkProtocol`, `KeyStatsInterfaceVersion`, `HelperErrorCode`.
- `KeyStats/Shared/HelperPayloadFields.swift` — Shared string constants for payload dictionary keys (`type`, `keyCode`, `flags`, `buttonRole`, `monotonicTime`, …).
- `KeyStats/Shared/HelperManifest.swift` — Codable struct + loader for `HelperManifest.json`; used by both `HelperSupervisor` (validate installed) and `build_helper.sh` (via a companion CLI or manual inspection).
- `KeyStats/Shared/HelperLocations.swift` — Hard-coded paths: Application Support install dir, LaunchAgent plist path, Mach service name, Label.

### New files (helper target)
- `KeyStatsHelper/main.swift` — Entry point: start XPC listener, call `CFRunLoopRun()`.
- `KeyStatsHelper/EventTapController.swift` — Creates / tears down CGEventTap, handles `DisabledByTimeout/UserInput`, 30Hz mouse sampling, pushes payloads to the active sink.
- `KeyStatsHelper/PayloadBuilder.swift` — Pure function `buildPayload(from:CGEvent, type:CGEventType)` → `[String: Any]` plist-safe.
- `KeyStatsHelper/ButtonRoleClassifier.swift` — Maps `CGEventType` + `NSEvent.subtype` + swap-preference → `primary / secondary / back / forward`.
- `KeyStatsHelper/HelperXPCListener.swift` — `NSXPCListenerDelegate`; applies `setConnectionCodeSigningRequirement`, validates peer via `SecCode*`, serializes to single active client.
- `KeyStatsHelper/HelperIdleSupervisor.swift` — Shuts the helper process down after N seconds of no connection.
- `KeyStatsHelper/Info.plist`
- `KeyStatsHelper/KeyStatsHelper.entitlements`

### New files (scripts & assets)
- `scripts/build_helper.sh`
- `scripts/verify_helper_payload.sh`
- `KeyStats/Resources/Helper/KeyStatsHelper.app.zip` — committed payload (binary, regenerated by `build_helper.sh` only when Helper source changes).
- `KeyStats/Resources/Helper/HelperManifest.json` — committed manifest.

### Modified files
- `KeyStats/InputMonitor.swift` — kept unchanged through Phase 4, deleted in Phase 7.
- `KeyStats/AppDelegate.swift` — permission/lifecycle routed through `HelperSupervisor` + `RemoteEventProcessor`.
- `KeyStats/AccessibilityPermissionCoordinator.swift` — `appURLs` points at installed helper URL.

## Testing Strategy

- **SPM unit tests** (`swift test`) for pure logic: manifest parsing, payload build/decode round-trip, button role classification, version negotiation. New test file per logic unit under `KeyStatsTests/`.
- **Manual verification** for system-level behavior: LaunchAgent bootstrap/bootout, XPC on-demand launch, peer validation, TCC behavior after update, quarantine handling. Each manual step lists the exact commands and expected output.
- **No UI tests** — project has none; we will not introduce any. Smoke tests happen by building in Xcode (⌘R) and observing menu-bar counts.
- **No mocking of NSXPCConnection** — ends up poorly testing transport realities. Prefer real launchctl/XPC smoke tests run manually.

```swift
import Foundation
import Cocoa
import CoreGraphics
import Carbon

/// Keyboard decoding extracted from InputMonitor so both the legacy tap
/// consumer and RemoteEventProcessor can translate key codes to names
/// without duplicating the TIS layout cache.
final class InputEventDecoder {
    static let shared = InputEventDecoder()

    private let layoutLock = NSLock()
    private var cachedLayoutData: CFData?
    private var inputSourceObserver: NSObjectProtocol?

    private init() {
        startInputSourceMonitoring()
        refreshKeyboardLayoutCache()
    }

    deinit {
        if let observer = inputSourceObserver {
            DistributedNotificationCenter.default().removeObserver(observer)
        }
    }

    /// Translate a keyDown CGEvent to a heatmap-friendly key name, including
    /// modifier prefixes (e.g. `LeftCmd+Shift+A`).
    func keyName(for event: CGEvent) -> String {
        let keyCode = Int(event.getIntegerValueField(.keyboardEventKeycode))
        let keyboardType = UInt32(event.getIntegerValueField(.keyboardEventKeyboardType))
        let base = baseKeyName(for: keyCode, keyboardType: keyboardType)
        let modifiers = keyboardEventModifierNames(rawFlags: event.flags.rawValue, keyCode: keyCode)
        if modifiers.isEmpty { return base }
        return modifiers.joined(separator: "+") + "+" + base
    }

    /// Same translation, but driven entirely by scalar fields so it can run
    /// on a helper payload (no CGEvent reference needed).
    func keyName(keyCode: Int, keyboardType: UInt32, flagsRaw: UInt64) -> String {
        let base = baseKeyName(for: keyCode, keyboardType: keyboardType)
        let modifiers = keyboardEventModifierNames(rawFlags: flagsRaw, keyCode: keyCode)
        if modifiers.isEmpty { return base }
        return modifiers.joined(separator: "+") + "+" + base
    }

    // MARK: - Base key name

    private func baseKeyName(for keyCode: Int, keyboardType: UInt32) -> String {
        if let mapped = Self.keyCodeMap[keyCode] {
            return mapped
        }
        if let asciiName = asciiKeyName(for: keyCode, keyboardType: keyboardType) {
            return asciiName
        }
        return "Key\(keyCode)"
    }

    private static let keyCodeMap: [Int: String] = [
        36: "Return", 48: "Tab", 49: "Space", 51: "Delete", 53: "Esc",
        63: "Fn", 64: "F17", 71: "Clear", 76: "Enter", 79: "F18",
        80: "F19", 96: "F5", 97: "F6", 98: "F7", 99: "F3",
        100: "F8", 101: "F9", 103: "F11", 105: "F13", 106: "F16",
        107: "F14", 109: "F10", 111: "F12", 113: "F15", 114: "Help",
        115: "Home", 116: "PageUp", 117: "ForwardDelete", 118: "F4",
        119: "End", 120: "F2", 121: "PageDown", 122: "F1", 123: "Left",
        124: "Right", 125: "Down", 126: "Up", 179: "Fn"
    ]

    // MARK: - Keyboard layout cache

    private func startInputSourceMonitoring() {
        let name = NSNotification.Name(kTISNotifySelectedKeyboardInputSourceChanged as String)
        inputSourceObserver = DistributedNotificationCenter.default().addObserver(
            forName: name,
            object: nil,
            queue: nil
        ) { [weak self] _ in
            self?.refreshKeyboardLayoutCache()
        }
    }

    private func refreshKeyboardLayoutCache() {
        let currentSource = TISCopyCurrentKeyboardLayoutInputSource()?.takeRetainedValue()
        let currentDataPtr = currentSource.flatMap { TISGetInputSourceProperty($0, kTISPropertyUnicodeKeyLayoutData) }
        var layoutData = currentDataPtr.map { Unmanaged<CFData>.fromOpaque($0).takeUnretainedValue() }
        if layoutData == nil {
            if let asciiSource = TISCopyCurrentASCIICapableKeyboardLayoutInputSource()?.takeRetainedValue() {
                let asciiDataPtr = TISGetInputSourceProperty(asciiSource, kTISPropertyUnicodeKeyLayoutData)
                layoutData = asciiDataPtr.map { Unmanaged<CFData>.fromOpaque($0).takeUnretainedValue() }
            }
        }
        layoutLock.lock()
        cachedLayoutData = layoutData
        layoutLock.unlock()
    }

    private func asciiKeyName(for keyCode: Int, keyboardType: UInt32) -> String? {
        layoutLock.lock()
        let layoutData = cachedLayoutData
        layoutLock.unlock()
        guard let layoutData = layoutData else { return nil }
        guard let layoutPtr = CFDataGetBytePtr(layoutData) else { return nil }
        let keyboardLayout = unsafeBitCast(layoutPtr, to: UnsafePointer<UCKeyboardLayout>.self)

        var deadKeyState: UInt32 = 0
        var chars = [UniChar](repeating: 0, count: 4)
        var actualLength: Int = 0
        let modifiers: UInt32 = 0
        let status = UCKeyTranslate(
            keyboardLayout,
            UInt16(keyCode),
            UInt16(kUCKeyActionDown),
            modifiers,
            keyboardType,
            UInt32(kUCKeyTranslateNoDeadKeysBit),
            &deadKeyState,
            chars.count,
            &actualLength,
            &chars
        )

        guard status == noErr, actualLength > 0 else { return nil }
        let raw = String(utf16CodeUnits: chars, count: actualLength)
        let cleaned = raw.trimmingCharacters(in: .whitespacesAndNewlines)
        guard !cleaned.isEmpty else { return nil }
        guard cleaned.count == 1 else { return nil }
        if cleaned == " " { return "Space" }
        if cleaned == "\t" { return "Tab" }
        if cleaned == "\r" { return "Return" }
        return cleaned.uppercased()
    }
}
```

```swift
import Foundation
import Cocoa
import CoreGraphics

enum ButtonRoleClassifier {
    private static let swapLeftRightButtonKey = "com.apple.mouse.swapLeftRightButton"

    /// Classify a mouse-button-down event into a semantic role. Returns nil
    /// for non-mouse events.
    static func role(for type: CGEventType, event: CGEvent) -> String? {
        switch type {
        case .leftMouseDown:
            return shouldSwap(for: event) ? ButtonRole.secondary : ButtonRole.primary
        case .rightMouseDown:
            return shouldSwap(for: event) ? ButtonRole.primary : ButtonRole.secondary
        case .otherMouseDown:
            let button = event.getIntegerValueField(.mouseEventButtonNumber)
            return button == 4 ? ButtonRole.forward : ButtonRole.back
        default:
            return nil
        }
    }

    private static func shouldSwap(for event: CGEvent) -> Bool {
        guard isPrimaryButtonRight() else { return false }
        guard let ns = NSEvent(cgEvent: event) else { return false }
        switch ns.subtype {
        case .mouseEvent: return true
        case .tabletPoint, .tabletProximity, .touch: return false
        default: return true
        }
    }

    private static func isPrimaryButtonRight() -> Bool {
        let key = swapLeftRightButtonKey as CFString
        if let v = CFPreferencesCopyValue(key, kCFPreferencesAnyApplication, kCFPreferencesAnyUser, kCFPreferencesCurrentHost) as? NSNumber {
            return v.boolValue
        }
        if let v = CFPreferencesCopyValue(key, kCFPreferencesAnyApplication, kCFPreferencesCurrentUser, kCFPreferencesAnyHost) as? NSNumber {
            return v.boolValue
        }
        if let v = UserDefaults.standard.object(forKey: swapLeftRightButtonKey) as? NSNumber {
            return v.boolValue
        }
        return false
    }
}
```

```swift
    /// Build a property-list-safe dictionary describing a CGEvent. Returns
    /// nil if the event type is not monitored.
    static func build(from event: CGEvent, type: CGEventType) -> [String: Any]? {
        var p: [String: Any] = [

            HelperPayloadFields.monotonicTime: NSNumber(value: ProcessInfo.processInfo.systemUptime),
            HelperPayloadFields.sourcePID: NSNumber(value: event.getIntegerValueField(.eventSourceUnixProcessID))
        ]

        switch type {
        case .keyDown, .flagsChanged:
            p[HelperPayloadFields.keyCode] = NSNumber(value: event.getIntegerValueField(.keyboardEventKeycode))
            p[HelperPayloadFields.keyboardType] = NSNumber(value: event.getIntegerValueField(.keyboardEventKeyboardType))
            p[HelperPayloadFields.flags] = NSNumber(value: event.flags.rawValue)
            p[HelperPayloadFields.isAutoRepeat] = NSNumber(value: event.getIntegerValueField(.keyboardEventAutorepeat) != 0)

        case .leftMouseDown, .rightMouseDown, .otherMouseDown:
```

```swift
import Foundation
import Cocoa
import CoreGraphics
import ApplicationServices

protocol EventPayloadSink: AnyObject {
    func forward(payload: [String: Any])
}

final class EventTapController {
    private var tap: CFMachPort?
    private var runLoopSource: CFRunLoopSource?
    private weak var sink: EventPayloadSink?

    private let mouseSampleInterval: TimeInterval = 1.0 / 30.0
    private var lastMouseSampleTime: TimeInterval = 0
    private let lock = NSLock()

    private static let eventMask: CGEventMask = {
        let keyboard = (1 << CGEventType.keyDown.rawValue) |
                       (1 << CGEventType.flagsChanged.rawValue)
        let mouse = (1 << CGEventType.leftMouseDown.rawValue) |
                    (1 << CGEventType.rightMouseDown.rawValue) |
                    (1 << CGEventType.otherMouseDown.rawValue) |
                    (1 << CGEventType.mouseMoved.rawValue) |
                    (1 << CGEventType.leftMouseDragged.rawValue) |
                    (1 << CGEventType.rightMouseDragged.rawValue) |
                    (1 << CGEventType.scrollWheel.rawValue)
        return CGEventMask(keyboard | mouse)
    }()

    func isAccessibilityGranted() -> Bool {
        AXIsProcessTrusted()
    }

    @discardableResult
    func start(sink: EventPayloadSink) -> Int {
        lock.lock(); defer { lock.unlock() }
        if tap != nil { return HelperErrorCode.alreadyMonitoring }
        guard isAccessibilityGranted() else { return HelperErrorCode.accessibilityDenied }
        self.sink = sink

        let ctx = Unmanaged.passUnretained(self).toOpaque()
        let cb: CGEventTapCallBack = { _, type, event, refcon in
            guard let refcon = refcon else { return Unmanaged.passUnretained(event) }
            let me = Unmanaged<EventTapController>.fromOpaque(refcon).takeUnretainedValue()
            me.handle(type: type, event: event)
            return Unmanaged.passUnretained(event)
        }

        guard let t = CGEvent.tapCreate(
            tap: .cgSessionEventTap,
            place: .headInsertEventTap,
            options: .listenOnly,
            eventsOfInterest: Self.eventMask,
            callback: cb,
            userInfo: ctx
        ) else { return HelperErrorCode.tapCreateFailed }

        tap = t
        runLoopSource = CFMachPortCreateRunLoopSource(kCFAllocatorDefault, t, 0)
        CFRunLoopAddSource(CFRunLoopGetMain(), runLoopSource, .commonModes)
        CGEvent.tapEnable(tap: t, enable: true)
        return HelperErrorCode.none
    }

    func stop() {
        lock.lock(); defer { lock.unlock() }
        if let t = tap { CGEvent.tapEnable(tap: t, enable: false) }
        if let s = runLoopSource { CFRunLoopRemoveSource(CFRunLoopGetMain(), s, .commonModes) }
        tap = nil
        runLoopSource = nil
        sink = nil
        lastMouseSampleTime = 0
    }

    // MARK: - Callback

    private func handle(type: CGEventType, event: CGEvent) {
        // System disables the tap on timeout / user input; re-enable.
        if type == .tapDisabledByTimeout || type == .tapDisabledByUserInput {
            if let t = tap { CGEvent.tapEnable(tap: t, enable: true) }
            return
        }

        // 30Hz mouse move sampling so we don't flood XPC.
        switch type {
        case .mouseMoved, .leftMouseDragged, .rightMouseDragged:
            let now = CFAbsoluteTimeGetCurrent()
            if now - lastMouseSampleTime < mouseSampleInterval { return }
            lastMouseSampleTime = now
        default:
            break
        }

        guard let payload = PayloadBuilder.build(from: event, type: type) else { return }
        sink?.forward(payload: payload)
    }
}
```

```swift
        case manifestDecodeFailed
        case unzipFailed(String)

    /// Installs if missing, upgrades in place if manifest cdhash differs.
    func ensureInstalled() throws {

        try FileManager.default.createDirectory(at: HelperLocations.installDir, withIntermediateDirectories: true)

        if existingCDHash == bundled.cdhash {
            return

        if fm.fileExists(atPath: HelperLocations.installedHelperURL.path) {
            try fm.removeItem(at: HelperLocations.installedHelperURL)
        }

    private func install(expected: HelperManifest) throws {
        let fm = FileManager.default
        let tmp = fm.temporaryDirectory.appendingPathComponent("KeyStatsHelperInstall-\(UUID().uuidString)", isDirectory: true)
        try fm.createDirectory(at: tmp, withIntermediateDirectories: true)
        defer { try? fm.removeItem(at: tmp) }

        guard let zipURL = bundledZipURL() else {
            throw SupervisorError.missingPayload
        }
        try unzip(zipURL, into: tmp)

        let unpackedApp = tmp.appendingPathComponent("KeyStatsHelper.app", isDirectory: true)
        try stripQuarantine(at: unpackedApp)
        try verifyCodeSignature(at: unpackedApp)

        let cdhash = try readCDHash(at: binaryURL(for: unpackedApp))
        guard cdhash == expected.cdhash else {
            throw SupervisorError.cdhashMismatch(expected: expected.cdhash, got: cdhash)
        }

        // Atomic replace.
        let target = HelperLocations.installedHelperURL
        if fm.fileExists(atPath: target.path) {
            try fm.removeItem(at: target)
        }
        try fm.moveItem(at: unpackedApp, to: target)
    }

    private func bundledZipURL() -> URL? {
        return Bundle.main.url(forResource: "KeyStatsHelper.app", withExtension: "zip", subdirectory: "Helper")
    }

    private func loadBundledManifest() throws -> HelperManifest {
        guard let url = Bundle.main.url(forResource: "HelperManifest", withExtension: "json", subdirectory: "Helper") else {
            throw SupervisorError.missingPayload
        }
        return try HelperManifest.load(from: url)
    }

    // MARK: - Helpers

    private func binaryURL(for appURL: URL) -> URL {
        appURL.appendingPathComponent("Contents/MacOS/KeyStatsHelper")
    }

    private func readInstalledCDHash() throws -> String {
        try readCDHash(at: HelperLocations.installedHelperBinaryURL)
    }

    private func readCDHash(at url: URL) throws -> String {
        var staticCode: SecStaticCode?
        let s = SecStaticCodeCreateWithPath(url as CFURL, [], &staticCode)
        guard s == errSecSuccess, let staticCode = staticCode else {
            throw SupervisorError.signatureInvalid(s)
        }
        var info: CFDictionary?
        let i = SecCodeCopySigningInformation(staticCode, SecCSFlags(rawValue: kSecCSDefaultFlags), &info)
        guard i == errSecSuccess, let d = info as? [String: Any], let data = d["cdhash"] as? Data else {
            throw SupervisorError.signatureInvalid(i)
        }
        return data.map { String(format: "%02x", $0) }.joined()
    }

    private func unzip(_ zip: URL, into dir: URL) throws {
        let p = Process()
        p.launchPath = "/usr/bin/ditto"
        p.arguments = ["-x", "-k", zip.path, dir.path]
        let err = Pipe()
        p.standardError = err
        try p.run()
        p.waitUntilExit()
        if p.terminationStatus != 0 {
            let msg = String(data: err.fileHandleForReading.readDataToEndOfFile(), encoding: .utf8) ?? ""
            throw SupervisorError.unzipFailed(msg)
        }
    }

    private func stripQuarantine(at url: URL) throws {
        let p = Process()
        p.launchPath = "/usr/bin/xattr"
        p.arguments = ["-rd", "com.apple.quarantine", url.path]
        try p.run()
        p.waitUntilExit()
        // Not fatal if xattr missing: the file may simply not have quarantine.
    }

    private func verifyCodeSignature(at appURL: URL) throws {
        var staticCode: SecStaticCode?
        let s = SecStaticCodeCreateWithPath(appURL as CFURL, [], &staticCode)
        guard s == errSecSuccess, let staticCode = staticCode else {
            throw SupervisorError.signatureInvalid(s)
        }
        let v = SecStaticCodeCheckValidity(staticCode, SecCSFlags(rawValue: kSecCSDefaultFlags), nil)
        if v != errSecSuccess {
            throw SupervisorError.signatureInvalid(v)
        }
    }
}
```

```swift
/// Exits the helper process after a period of no active XPC connection.
final class HelperIdleSupervisor {

    private var timer: DispatchSourceTimer?
    private let queue = DispatchQueue(label: "com.keystats.app.helper.idle")

    func noteActivity() {
        schedule()
    }

        schedule()
    }

    private func schedule() {
        queue.async { [weak self] in
            guard let self = self else { return }
            self.timer?.cancel()
            let t = DispatchSource.makeTimerSource(queue: self.queue)
            t.schedule(deadline: .now() + self.idleTimeout)
            t.setEventHandler {
                NSLog("[KeyStatsHelper] idle timeout, exiting")
                exit(0)
            }
            t.resume()
            self.timer = t
        }
    }
```

```swift
import Foundation
import Security
import AppKit

/// XPC listener with peer validation. Accepts at most one active connection.
/// Anything else is rejected so Helper never exposes an event stream to two
/// clients simultaneously.
final class HelperXPCListener: NSObject, NSXPCListenerDelegate, KeyStatsHelperProtocol, EventPayloadSink {
    private let listener: NSXPCListener
    private let idle = HelperIdleSupervisor()
    private let tap = EventTapController()
    private let stateLock = NSLock()
    private weak var activeConnection: NSXPCConnection?
    private weak var activeSink: KeyStatsEventSinkProtocol?

    override init() {
        self.listener = NSXPCListener(machServiceName: HelperLocations.machServiceName)
        super.init()
        self.listener.delegate = self

        if #available(macOS 13.0, *) {
            // First-layer pre-filter. Accepts any code signed with the main-app
            // identifier. Unsigned or wrong-identifier peers are rejected by
            // the system before delegate runs.
            let req = "identifier \"\(HelperLocations.mainBundleId)\""
            self.listener.setConnectionCodeSigningRequirement(req)
        }
    }

    func resume() { listener.resume() }

    // MARK: - Delegate

    func listener(_ listener: NSXPCListener,
                  shouldAcceptNewConnection newConnection: NSXPCConnection) -> Bool {
        guard validatePeer(newConnection) else {
            NSLog("[KeyStatsHelper] peer validation failed, rejecting")
            return false
        }

        stateLock.lock()
        if activeConnection != nil {
            stateLock.unlock()
            NSLog("[KeyStatsHelper] active connection exists, rejecting new")
            return false
        }
        stateLock.unlock()

        newConnection.exportedInterface = NSXPCInterface(with: KeyStatsHelperProtocol.self)
        newConnection.exportedObject = self

        let remote = NSXPCInterface(with: KeyStatsEventSinkProtocol.self)
        let payloadClasses: Set<AnyHashable> = [
            NSDictionary.self, NSString.self, NSNumber.self
        ]
        remote.setClasses(
            NSSet(set: payloadClasses) as! Set<AnyHashable>,
            for: #selector(KeyStatsEventSinkProtocol.receiveEvent(_:)),
            argumentIndex: 0,
            ofReply: false
        )
        newConnection.remoteObjectInterface = remote

        newConnection.invalidationHandler = { [weak self, weak newConnection] in
            self?.handleConnectionClosed(newConnection)
        }
        newConnection.interruptionHandler = { [weak self, weak newConnection] in
            self?.handleConnectionClosed(newConnection)
        }

        stateLock.lock()
        activeConnection = newConnection
        activeSink = newConnection.remoteObjectProxy as? KeyStatsEventSinkProtocol
        stateLock.unlock()

        idle.noteActivity()
        newConnection.resume()
        return true
    }

    private func validatePeer(_ conn: NSXPCConnection) -> Bool {
        let pid = conn.processIdentifier
        guard pid > 0 else { return false }

        var code: SecCode?
        let attrs: [String: Any] = [kSecGuestAttributePid as String: Int(pid)]
        var status = SecCodeCopyGuestWithAttributes(nil, attrs as CFDictionary, [], &code)
        guard status == errSecSuccess, let code = code else { return false }

        status = SecCodeCheckValidity(code, [], nil)
        guard status == errSecSuccess else { return false }

        var info: CFDictionary?
        status = SecCodeCopySigningInformation(
            code as! SecStaticCode,
            SecCSFlags(rawValue: kSecCSSigningInformation),
            &info
        )
        guard status == errSecSuccess, let d = info as? [String: Any] else { return false }

        guard let identifier = d[kSecCodeInfoIdentifier as String] as? String,
              identifier == HelperLocations.mainBundleId else {
            return false
        }

        // Session + user checks.
        let uid = conn.effectiveUserIdentifier
        guard uid == getuid() else { return false }

        return true
    }

    private func handleConnectionClosed(_ conn: NSXPCConnection?) {
        stateLock.lock()
        if activeConnection === conn {
            activeConnection = nil
            activeSink = nil
            tap.stop()
        }
        stateLock.unlock()
        idle.connectionDidClose()
    }

    // MARK: - KeyStatsHelperProtocol

    func handshake(clientInterfaceVersion: Int,
                   reply: @escaping (Int, Bool) -> Void) {
        idle.noteActivity()
        reply(HelperLocations.interfaceVersion, tap.isAccessibilityGranted())
    }

    func startMonitoring(reply: @escaping (Bool, Int) -> Void) {
        idle.noteActivity()
        let code = tap.start(sink: self)
        reply(code == HelperErrorCode.none, code)
    }

    func stopMonitoring() {
        idle.noteActivity()
        tap.stop()
    }

    func helperBundleURL(reply: @escaping (String) -> Void) {
        reply(Bundle.main.bundleURL.path)
    }

    // MARK: - EventPayloadSink

    func forward(payload: [String: Any]) {
        stateLock.lock()
        let sink = activeSink
        stateLock.unlock()
        sink?.receiveEvent(payload)
    }
}
```

```swift
import Foundation

/// Thin wrapper over NSXPCConnection. Single-active-connection semantics
/// mirror the Helper's listener. Reconnects on invalidation with backoff.
final class HelperXPCClient {
    static let shared = HelperXPCClient()

    enum State: Equatable {
        case idle
        case connecting
        case connected(helperVersion: Int, accessibilityGranted: Bool)
        case disconnected(reason: String)
    }

    private let lock = NSLock()
    private var connection: NSXPCConnection?
    private var _state: State = .idle
    private var eventSink: KeyStatsEventSinkProtocol?
    private var stateObservers: [(State) -> Void] = []

    private init() {}

    var state: State {
        lock.lock(); defer { lock.unlock() }
        return _state
    }

    func setEventSink(_ sink: KeyStatsEventSinkProtocol) {
        lock.lock(); defer { lock.unlock() }
        self.eventSink = sink
        rewireSinkIfConnected_locked()
    }

    func addStateObserver(_ cb: @escaping (State) -> Void) {
        lock.lock(); stateObservers.append(cb); lock.unlock()
        cb(state)
    }

    /// Idempotent: create connection if missing, send handshake, resolve state.
    func connect(completion: ((State) -> Void)? = nil) {
        lock.lock()
        if connection != nil {
            let s = _state
            lock.unlock()
            completion?(s)
            return
        }
        let c = NSXPCConnection(machServiceName: HelperLocations.machServiceName)
        c.remoteObjectInterface = NSXPCInterface(with: KeyStatsHelperProtocol.self)
        let exported = NSXPCInterface(with: KeyStatsEventSinkProtocol.self)
        let classes: Set<AnyHashable> = [NSDictionary.self, NSString.self, NSNumber.self]
        exported.setClasses(
            classes as! Set<AnyHashable>,
            for: #selector(KeyStatsEventSinkProtocol.receiveEvent(_:)),
            argumentIndex: 0,
            ofReply: false
        )
        c.exportedInterface = exported
        if let sink = eventSink {
            c.exportedObject = sink
        }
        c.invalidationHandler = { [weak self] in
            self?.transition(to: .disconnected(reason: "invalidated"))
        }
        c.interruptionHandler = { [weak self] in
            self?.transition(to: .disconnected(reason: "interrupted"))
        }
        c.resume()
        connection = c
        _state = .connecting
        lock.unlock()

        let proxy = c.remoteObjectProxyWithErrorHandler { [weak self] err in
            self?.transition(to: .disconnected(reason: "\(err)"))
        } as? KeyStatsHelperProtocol
        guard let proxy = proxy else {
            transition(to: .disconnected(reason: "no proxy"))
            completion?(.disconnected(reason: "no proxy"))
            return
        }
        proxy.handshake(clientInterfaceVersion: HelperLocations.interfaceVersion) { [weak self] helperVersion, granted in
            let newState: State = .connected(helperVersion: helperVersion, accessibilityGranted: granted)
            self?.transition(to: newState)
            completion?(newState)
        }
    }

    func startMonitoring(completion: @escaping (Bool, Int) -> Void) {
        withProxy { proxy in
            proxy.startMonitoring(reply: completion)
        } fallback: {
            completion(false, HelperErrorCode.accessibilityDenied)
        }
    }

    func stopMonitoring() {
        withProxy { $0.stopMonitoring() } fallback: {}
    }

    func disconnect() {
        lock.lock()
        let c = connection
        connection = nil
        _state = .idle
        lock.unlock()
        c?.invalidate()
    }

    // MARK: - Private

    private func withProxy<T>(_ fn: (KeyStatsHelperProtocol) -> T,
                              fallback: () -> T) -> T {
        lock.lock()
        let c = connection
        lock.unlock()
        guard let c = c else { return fallback() }
        let proxy = c.remoteObjectProxyWithErrorHandler { [weak self] err in
            self?.transition(to: .disconnected(reason: "\(err)"))
        } as? KeyStatsHelperProtocol
        return proxy.map(fn) ?? fallback()
    }

    private func rewireSinkIfConnected_locked() {
        if let sink = eventSink {
            connection?.exportedObject = sink
        }
    }

    private func transition(to state: State) {
        var observers: [(State) -> Void] = []
        lock.lock()
        _state = state
        observers = stateObservers
        if case .disconnected = state {
            connection?.invalidate()
            connection = nil
        }
        lock.unlock()
        observers.forEach { $0(state) }
    }
}
```

```swift
import Foundation
import CoreGraphics
import Cocoa

final class RemoteEventProcessor: NSObject, KeyStatsEventSinkProtocol {
    static let shared = RemoteEventProcessor()

    private let decoder = InputEventDecoder.shared
    private var modifierTracker = ModifierStandaloneTracker()
    private let modifierLock = NSLock()

    private override init() { super.init() }

    func receiveEvent(_ payload: [String: Any]) {
        guard let typeRaw = (payload[HelperPayloadFields.type] as? NSNumber)?.uint32Value,
              let type = CGEventType(rawValue: typeRaw) else { return }

        let monotonic = (payload[HelperPayloadFields.monotonicTime] as? NSNumber)?.doubleValue ?? 0
        let pid = (payload[HelperPayloadFields.sourcePID] as? NSNumber)?.int32Value ?? 0
        let stats = StatsManager.shared

        switch type {
        case .keyDown:
            guard let isAutoRepeat = (payload[HelperPayloadFields.isAutoRepeat] as? NSNumber)?.boolValue,
                  !isAutoRepeat else { return }
            if stats.handleMouseDistanceCalibrationKeyPress() { return }
            guard let keyCode = (payload[HelperPayloadFields.keyCode] as? NSNumber)?.intValue,
                  let keyboardType = (payload[HelperPayloadFields.keyboardType] as? NSNumber)?.uint32Value,
                  let flagsRaw = (payload[HelperPayloadFields.flags] as? NSNumber)?.uint64Value
            else { return }

            modifierLock.lock()
            modifierTracker.consumePendingModifiers(forKeyDownWith: flagsRaw, keyCode: keyCode)
            modifierLock.unlock()

            let name = decoder.keyName(keyCode: keyCode, keyboardType: keyboardType, flagsRaw: flagsRaw)
            stats.incrementKeyPresses(keyName: name, appIdentity: appIdentity(forPID: pid))

        case .flagsChanged:
            guard let keyCode = (payload[HelperPayloadFields.keyCode] as? NSNumber)?.intValue,
                  let flagsRaw = (payload[HelperPayloadFields.flags] as? NSNumber)?.uint64Value
            else { return }

            if keyCode == 57 {
                stats.incrementKeyPresses(keyName: "CapsLock", appIdentity: appIdentity(forPID: pid))
                return
            }

            modifierLock.lock()
            let modifierName = modifierTracker.handleFlagsChanged(keyCode: keyCode, rawFlags: flagsRaw)
            modifierLock.unlock()
            if let modifierName = modifierName {
                stats.incrementKeyPresses(keyName: modifierName, appIdentity: appIdentity(forPID: pid))
            }

        case .leftMouseDown, .rightMouseDown, .otherMouseDown:
            let role = payload[HelperPayloadFields.buttonRole] as? String
            let ident = appIdentity(forPID: pid)
            switch role {
            case ButtonRole.primary: stats.incrementLeftClicks(appIdentity: ident)
            case ButtonRole.secondary: stats.incrementRightClicks(appIdentity: ident)
            case ButtonRole.back: stats.incrementSideBackClicks(appIdentity: ident)
            case ButtonRole.forward: stats.incrementSideForwardClicks(appIdentity: ident)
            default: break
            }

        case .scrollWheel:
            let dx = (payload[HelperPayloadFields.scrollDX] as? NSNumber)?.doubleValue ?? 0
            let dy = (payload[HelperPayloadFields.scrollDY] as? NSNumber)?.doubleValue ?? 0
            let total = sqrt(dx * dx + dy * dy)
            stats.addScrollDistance(total * 10, appIdentity: appIdentity(forPID: pid))

        case .mouseMoved, .leftMouseDragged, .rightMouseDragged:
            guard let x = (payload[HelperPayloadFields.locationX] as? NSNumber)?.doubleValue,
                  let y = (payload[HelperPayloadFields.locationY] as? NSNumber)?.doubleValue
            else { return }
            let current = CGPoint(x: x, y: y)
            if let last = stats.lastMousePosition {
                let dx = current.x - last.x
                let dy = current.y - last.y
                let d = sqrt(dx * dx + dy * dy)
                if d < 500 {
                    if stats.isMouseDistanceCalibrating {
                        stats.recordMouseDistanceCalibration(d)
                    } else {
                        stats.addMouseDistance(d)
                    }
                }
            }
            stats.lastMousePosition = current

        default:
            break
        }

        _ = monotonic // reserved for StatsManager monotonic time sources in Task 4.4
    }

    private func appIdentity(forPID pid: pid_t) -> AppIdentity? {
        guard StatsManager.shared.appStatsEnabled else { return nil }
        return AppActivityTracker.shared.appIdentity(forPID: pid)
    }
}
```

```swift
private func startPermissionPolling() {
    permissionCheckTimer?.invalidate()
    permissionCheckCount = 0
    permissionCheckTimer = Timer.scheduledTimer(withTimeInterval: 2.0, repeats: true) { [weak self] timer in
        guard let self = self else { timer.invalidate(); return }
        self.permissionCheckCount += 1

        if self.isUsingHelperPipeline {
            HelperXPCClient.shared.connect { state in
                if case .connected(_, let granted) = state, granted {
                    DispatchQueue.main.async {
                        timer.invalidate()
                        self.permissionCheckTimer = nil
                        self.promptLaunchAtLoginIfNeeded()
                        HelperXPCClient.shared.startMonitoring { _, _ in }
                        Self.trackEvent("permission_granted", properties: ["permission": "accessibility"])
                    }
                }
            }
        } else if InputMonitor.shared.hasAccessibilityPermission() {
            timer.invalidate()
            self.permissionCheckTimer = nil
            self.handleAccessibilityPermissionGranted()
            Self.trackEvent("permission_granted", properties: ["permission": "accessibility"])
        }

        if self.permissionCheckCount >= self.maxPermissionChecks {
            timer.invalidate()
            self.permissionCheckTimer = nil
        }
    }
    if let permissionCheckTimer {
        RunLoop.main.add(permissionCheckTimer, forMode: .common)
    }
}
```

```swift
import PermissionFlow

extension NSView {
    func permissionFlowSourceFrameInScreen() -> CGRect? {
        guard let window = window else { return nil }
        let frameInWindow = convert(bounds, to: nil)
        return window.convertToScreen(frameInWindow)
    }
}

@MainActor
final class AccessibilityPermissionCoordinator {
    static let shared = AccessibilityPermissionCoordinator()

    private let controller: PermissionFlowController

    private init() {
        self.controller = PermissionFlow.makeController(
            configuration: .init(
                requiredAppURLs: Self.appURLsForActivePipeline(),
                promptForAccessibilityTrust: false
            )
        )
    }

    private static func appURLsForActivePipeline() -> [URL] {
        if InputPipelineConfig.useHelper {
            let url = HelperLocations.installedHelperURL
            if FileManager.default.fileExists(atPath: url.path) {
                return [url]
            }
        }
        return [Bundle.main.bundleURL]
    }

    func requestPermission(sourceFrameInScreen: CGRect? = nil) {
        let urls = Self.appURLsForActivePipeline()
        controller.authorize(
            pane: .accessibility,
            suggestedAppURLs: urls,
            sourceFrameInScreen: sourceFrameInScreen ?? fallbackSourceFrameInScreen()
        )
    }

    private func fallbackSourceFrameInScreen() -> CGRect {
        let mouseLocation = NSEvent.mouseLocation
        return CGRect(x: mouseLocation.x - 16, y: mouseLocation.y - 16, width: 32, height: 32)
    }
}
```
